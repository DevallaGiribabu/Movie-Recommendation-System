# 🎬 Movie Recommendation System

An AI-based movie recommendation system using Python and Machine Learning.

## Features

- Recommend similar movies
- Content-Based Filtering
- Cosine Similarity
- User movie input

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn

## Run Project

Install packages:

pip install -r requirements.txt

Run:

python movie_recommend.py
